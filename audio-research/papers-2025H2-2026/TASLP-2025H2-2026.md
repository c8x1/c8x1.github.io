### 【非编创】检索过程与各入口结果

本次严格遵循【全量枚举 + arXiv ID 反验】方法论，逐入口尝试：

1. **DBLP（首选全量枚举入口）**
   - `dblp.org/db/journals/taslp/` 期刊主页：仅列至 Volume 32 (2024)，**Volume 33 (2025) / Volume 34 (2026) 页面为 HTTP 404**（`taslp33.html`、`taslp34.html` 均返回 `dblp: error 404`）。
   - 即 DBLP 尚未为 2025/2026 卷建立索引页，无法逐 issue 全量枚举。
   - 作为对照，vol 32 (2024) 已收录 393 篇条目，结构正常；说明缺口在 vol 33+，而非方法错误。

2. **IEEE Xplore（官方 TOC，应为权威源）**
   - `curl` 直连与走代理（`-x http://127.0.0.1:7897`）访问 `ieeexplore.ieee.org/rest/pubsvc/journals/6570656/issues` 等 REST 端点：均返回 **HTTP 418** + 反爬启动页（Cloudflare/IEEE bot 拦截）。
   - WebFetch 抓 `ieeexplore.ieee.org`：返回 "Unable to verify if domain ieeexplore.ieee.org is safe to fetch"（环境网络策略拦截）。
   - 即本环境无法直接枚举 IEEE Xplore 上 vol 33/34 的 early-access / 已发表论文。punumber=6570656（TASLP 正确期刊 ID）；WebSearch 给出的 punumber=1032924 为幻觉，已排除。

3. **Crossref（DOI 权威注册库，按 ISSN 2329-9290 / 2329-9304）**
   - 按 `from-created-date:2025-07-01,until-created-date:2026-06-30` 过滤：**total-results = 0**。
   - 按 `from-pub-date:2025-07-01` 过滤：**0**。
   - 最新注册 DOI 为 `10.1109/taslp.2024.3520736`（"List of Reviewers"，created 2025-01-08）；最后一篇研究论文注册于 2024-11-28（`10.1109/taslp.2024.3507560`，IPDnet，vol 32）。
   - 结论：**IEEE 在 2025-07-01 之后未向 Crossref 注册任何 TASLP DOI**。这是元数据登记滞后，不代表 IEEE 未发表，但使 Crossref 无法用于本窗口枚举。

4. **OpenAlex（按 source id S4210169297）**
   - `publication_year:2025` / `2026` / `2025|2026` 三种过滤：均 **count = 0**。
   - 按 `publication_date:desc` 取最新 200 篇：全部 publication_year=2024、publication_date=2024-01-01（OpenAlex 未存月度精确期号日期）。
   - 结论：OpenAlex 同样存在索引滞后，最新条目止于 2024 卷 32。

5. **Semantic Scholar Graph API**
   - 全程 **429 Too Many Requests**（直连与代理均无效，符合技能文档"账号/IP 段级限流、换代理无效"踩坑记录），无法用作枚举/反验。

6. **arXiv（预印本反向核实入口）**
   - `all:TASLP` 检索（按提交时间倒序）返回 40 条 2025-07 至 2026-06 提交的语音/音频预印本，其 arXiv ID 均经 HTML abs 页（`arxiv.org/abs/<id>`，方法 B）逐条核实标题，ID 全部真实匹配（如 2606.17126、2604.14606、2509.08379、2508.16332 等）。

- **本沙箱环境无法完成 vol 33/34 的全量枚举**：DBLP 未建索引页、IEEE Xplore 被网络封锁（curl 418 + WebFetch 域名拒验）、Crossref/OpenAlex 元数据滞后（最新止于 2024-11/2025-01 注册）、S2 限流。
