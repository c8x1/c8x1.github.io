推理与产物出口

目标：讲清训练 dcp→merge→HF→推理 的产物出口链。前置：EP17 Checkpoint 与权重转换。

你将获得：① 四个推理文件的逐段精读——inference_fsdp2.py 如何像 train_fsdp2.py 一样起 NPU/HCCL、用 ModelFactory.create 加载合并后的 HF 权重并交给 Inferencer；chat_model.py 如何用工厂选 HuggingfaceEngine 并在后台 asyncio 线程里桥接同步 generate 与流式 stream_chat；hf_engine.py 如何用 transformers 的 apply_chat_template/TextIteratorStreamer 调 model.generate()、并强制 do_sample=False 防 FSDP 采样死锁；inferencer.py 如何在多 rank 下用 broadcast_object_list 把 rank0 的键盘输入广播给所有卡同步进 while True 聊天循环，每段贴真实 file:line ② 一个反直觉点——训练 dcp 不能直接喂推理：dcp 存的是每 rank 的分片 + DTensor 布局元信息，推理入口加载的是 EP17 merge 出来的 HF 全量 safetensors，这条"训练→推理"的衔接点完全压在 EP17 的 merge_dcp_to_hf 上；model.generate() 那一行（hf_engine.py:100）才是你熟悉的旧大陆被这条新链路最后接回去的地方。

你已经熟悉的情况 桥接开场

你写过 outputs = model.generate(input_ids, max_new_tokens=512)。单卡 GPU 世界里推理很简单：AutoModelForCausalLM.from_pretrained("Qwen/Qwen3-8B") 从 HF Hub 拉一份全量 safetensors、整份 mmap 到一张卡、model.eval()、torch.no_grad()、然后 model.generate() 自回归吐 token——KV cache 在框架内部按 past_key_values 维护，你不用操心分片。想做成聊天就再包一层 apply_chat_template 把 messages 拼成 prompt、配一个 TextIteratorStreamer + 后台线程做打字机效果——这套 HuggingFace transformers 的"老三样"你很熟。

问题来了：本项目训练完的权重不是全量 safetensors。EP08 讲过 FSDP2 下没有任何一张卡持有完整模型，EP16 讲过优化器 m/v 只跟本地分片，EP17 讲过训练期落盘走的是 dcp 分片 ckpt（每 rank 一份 __RANK_0.distcp + 一份 metadata）。而推理入口 inference_fsdp2.py:67 调 ModelFactory.create、再下钻到 model_loader.py:129 的 AutoModelForCausalLM.from_pretrained——这一步只认 HF 全量 safetensors，不认 dcp。中间这道鸿沟，就是 EP17 的 merge_dcp_to_hf 负责填的：它把 dcp 的分片读 metadata、按 shard 贪心装箱、逐 shard full_tensor() 拼回全量、写成 model-0000X-of-0000Y.safetensors + model.safetensors.index.json——产物格式与 HF save_pretrained 完全同构。所以本篇要厘清的关键认知是：推理加载的是"merge 后的 HF"，不是"训练里的 dcp"，model.generate() 那一行在两个世界里长得一模一样，但它脚底下的权重来源完全不同。

另一处熟悉又陌生的点是多卡推理。单卡世界你不必关心 rank，多卡 FSDP2 推理里每张卡只有 1/N 的参数（和训练一样的分片），model.generate 在 FSDP 包装下跑时各 rank 必须步调一致——否则一个 rank 采样到 EOS 提前退出、别的 rank 还在算，集合通信就对不上、死锁。hf_engine.py:78 的 do_sample=False（贪婪解码、所有 rank 同一输入同一输出）和 :86 的 synced_gpus=False 就是围绕这个的——这两个参数你在单卡从来不设，多卡 FSDP 推理里却成了硬约束。下面把整条产物出口链摊开读。

本项目怎么做 原理

核心原理一句话：推理是"merge 后用 HF 引擎跑"。训练侧产物（dcp 分片）不能直接喂推理，必须先经 EP17 的 merge_dcp_to_hf 合并成 HF 全量 safetensors——这一步把"训练的并行格式"翻译回"推理的通用格式"。合并后的目录结构跟从 HF Hub 下载的一模一样：config.json、model.safetensors.index.json、若干 model-0000X-of-0000Y.safetensors、tokenizer.json 等。推理入口 inference_fsdp2.py 把这个目录路径喂给 ModelFactory.create，后者走 ModelLoader 用 transformers 的 AutoModelForCausalLM.from_pretrained 加载——加载时模型还是全量在 CPU/meta 上，加载完才被 FSDP 的 fully_shard 重新切成每卡 1/N（训练切一次、加载再切一次，两次切分的拓扑可以不同，因为中间经过的全量 HF 是拓扑无关的中立格式）。模型就绪后，AutoInferencer 把 (model, tokenizer, args) 交给应用层 Inferencer，后者持一个 ChatModel，ChatModel 按参数选 HuggingfaceEngine（目前只有这一种 backend）并在后台 asyncio 线程里把同步的 model.generate 包装成异步的 chat/stream_chat。inferencer.run_interactive_chat 进 while True 循环：rank0 读键盘、broadcast_object_list 广播给所有 rank、各 rank 用同样的 messages 调 stream_chat、rank0 打印流式 token。

这条链路的关键衔接点压在 EP17 的 merge 上——这是"训练→推理"之间唯一的桥。merge 失败（比如忘传 --model-configs 导致产物目录没有 config.json），from_pretrained 直接报错，推理根本起不来。下图把整条产物出口流摊开。

注意图里那条红色虚线——① 的 dcp 不能跳过 ② 直接进 ④，这是本篇的核心误区。② 的 HF 全量是训练和推理之间唯一的中立中转格式（EP17 已述），inference_fsdp2.py 加载的就是 ② 的产物。下面把四个文件逐段拆开。

class AutoInferencer:
    """
    Responsible for setting up the runtime environment: NPU initialization,
    distributed setup, and loading the FSDP model.
    """
    def __init__(self):
        # 1. Parse arguments
        root_args = fsdp2_parse_args(Arguments)
        self.model_args = root_args.model
        self.parallel_args = root_args.parallel
        self.inference_args = root_args.inference
        self.args = types.SimpleNamespace(**{...})

        set_args(self.args)

        # 2. Initialize NPU and distributed environment
        self._initialize()

        # 3. Build Tokenizer
        logger.info_rank0("> Building Tokenizer...")
        self.tokenizer = TokenizerFactory.create(self.model_args)

        # 4. Build Model (FSDP automatic sharding strategies take effect here)
        # Force disable recomputation during inference to save overhead
        self.parallel_args.recompute = False
        logger.info_rank0("> Building Model for Inference...")

        # The model returned here is already FSDP-wrapped, each card only holds its own shard
        self.model = ModelFactory.create(self.model_args, self.parallel_args)

        # 5. Instantiate the application-level Inferencer
        self.inferencer = Inferencer(
            model=self.model,
            tokenizer=self.tokenizer,
            args=self.inference_args
        )

    @staticmethod
    def _initialize():
        """Initialize underlying hardware and distributed environment."""
        if is_torch_npu_available():
            fallback = torch.npu
            dist_backend = "hccl"
            apply_hccl_premul_sum_patch()
        elif torch.cuda.is_available():
            fallback = torch.cuda
            dist_backend = "nccl"
        set_accelerator_compatible(fallback)
        local_rank = int(os.environ.get("LOCAL_RANK", 0))
        torch.accelerator.set_device_index(local_rank)
        if not torch.distributed.is_initialized():
            torch.distributed.init_process_group(backend=dist_backend)

    def chat(self):
        """Launch interactive chat."""
        self.inferencer.run_interactive_chat()

def _create_on_cpu(self, model_cls=None) -> Tuple[nn.Module, None]:
    """Create and load model on CPU."""
    if model_cls is not None:
        logger.info_rank0(f"> Loading {model_cls.__name__} on CPU...")
        model = model_cls.from_pretrained(
            self.model_path,
            config=self.hf_config,
            low_cpu_mem_usage=True,
            device_map="cpu",
            torch_dtype=torch.float32,
        )
    elif self.train_from_scratch:
        logger.info_rank0("> Creating model with random weights on CPU...")
        model = AutoModelForCausalLM.from_config(
            self.hf_config, trust_remote_code=self.trust_remote_code, torch_dtype=torch.float32
        )
    else:
        logger.info_rank0(f"> Loading pretrained model on CPU from {self.model_path}...")
        model = AutoModelForCausalLM.from_pretrained(
            self.model_path,
            config=self.hf_config,
            trust_remote_code=self.trust_remote_code,
            torch_dtype=torch.float32,
            low_cpu_mem_usage=True,
            device_map="cpu",
        )
    return model, None

class HuggingfaceEngine(BaseEngine):
    def __init__(self, model, tokenizer, args) -> None:
        self.name = "huggingface"
        self.can_generate = True

        self.model = getattr(model, "model", model)
        self.tokenizer = tokenizer
        self.args = args

        if self.tokenizer.pad_token_id is None:
            self.tokenizer.pad_token_id = self.tokenizer.eos_token_id

        self.model.eval()
        # Initialize asyncio loop safely
        try:
            asyncio.get_running_loop()
        except RuntimeError:
            asyncio.set_event_loop(asyncio.new_event_loop())
        self.semaphore = asyncio.Semaphore(int(os.getenv("MAX_CONCURRENT", "1")))

    @staticmethod
    def _process_args(model, tokenizer, args, messages, **input_kwargs):
        target_device = HuggingfaceEngine._get_target_device()
        # Build prompt using chat template or fallback to raw text
        try:
            tokenizer_output = tokenizer.apply_chat_template(
                messages, add_generation_prompt=True, return_tensors="pt"
            ).to(target_device)
        except Exception:
            raw_text = "\n".join([m["content"] for m in messages])
            input_ids = tokenizer(raw_text, return_tensors="pt").input_ids.to(target_device)
        ...
        gen_config = model.generation_config
        gen_config.max_new_tokens = getattr(args, "max_new_tokens", 512)
        gen_config.do_sample = getattr(args, "do_sample", False)  # Disabled to prevent FSDP sampling deadlocks

mindspeed_llm/fsdp2/inference/chat_model.py —— ChatModel：工厂选 engine + 后台 asyncio 线程桥接同步/异步

class InferBackend(str, Enum):
    HUGGINGFACE = "huggingface"


class ChatModel:
    r"""General class for chat models. Bridges Async engine."""

    def __init__(self, model, tokenizer, args) -> None:
        # 1. Get the inference backend from args, default to huggingface
        infer_backend = getattr(args, "infer_backend", InferBackend.HUGGINGFACE).lower()

        # 2. Factory pattern to select the appropriate inference engine
        if infer_backend == InferBackend.HUGGINGFACE:
            self.engine: BaseEngine = HuggingfaceEngine(model, tokenizer, args)
        else:
            raise ValueError(f"Unknown infer_backend: {infer_backend}. Only choose from: {InferBackend.HUGGINGFACE}.")

        # 3. Start the background asyncio event loop
        self._loop = asyncio.new_event_loop()
        self._thread = Thread(target=_start_background_loop, args=(self._loop,), daemon=True)
        self._thread.start()

    def chat(self, messages, **input_kwargs) -> list["Response"]:
        task = asyncio.run_coroutine_threadsafe(
            self.engine.chat(messages, **input_kwargs), self._loop
        )
        return task.result()

    def stream_chat(self, messages, **input_kwargs) -> Generator[str, None, None]:
        generator = self.engine.stream_chat(messages, **input_kwargs)
        while True:
            try:
                task = asyncio.run_coroutine_threadsafe(generator.__anext__(), self._loop)
                yield task.result()
            except StopAsyncIteration:
                break

class Inferencer:
    """
    Executes inference tasks, backed by an extensible ChatModel engine.
    """
    def __init__(self, model, tokenizer, args):
        self.chat_model = ChatModel(model, tokenizer, args)
        self.rank = dist.get_rank() if dist.is_initialized() else 0

    def run_interactive_chat(self):
        """Starts an interactive chat loop (CLI Demo) with streaming support."""
        logger.info_rank0(">>> Entering Interactive Chat Mode. Type 'exit' to quit.")
        history = []
        EXIT_COMMANDS = ("exit", "quit")
        ROLE_USER, ROLE_ASSISTANT = ("user", "assistant")
        ASSISTANT_PREFIX = "Assistant: "
        VISUAL_SEPARATOR = "\n" + "-" * 40
        while True:
            # 1. Sync input across all FSDP ranks to prevent deadlocks
            user_input = self._get_sync_input()
            if user_input.lower() in EXIT_COMMANDS:
                break
            if not user_input:
                continue
            history.append({"role": ROLE_USER, "content": user_input})
            if self.rank == 0:
                print(ASSISTANT_PREFIX, end="", flush=True)
            response = ""
            # 2. Use the streaming API for the typewriter effect
            for new_text in self.chat_model.stream_chat(history):
                if self.rank == 0:
                    print(new_text, end="", flush=True)
                response += new_text
            logger.info_plain_rank0(VISUAL_SEPARATOR)
            history.append({"role": ROLE_ASSISTANT, "content": response})

    def _get_sync_input(self):
        """Only Rank 0 reads keyboard; broadcast to other Ranks for sync collective comm."""
        if self.rank == 0:
            try:
                user_input = input("\nUser: ").strip()
            except EOFError:
                user_input = "exit"
        else:
            user_input = None
        if dist.is_initialized():
            objects = [user_input]
            dist.broadcast_object_list(objects, src=0)
            user_input = objects[0]
        return user_input
