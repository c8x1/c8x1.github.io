左侧把 torch_npu.npu.* 全替换成 torch.cuda.*（CUDA 的 Event/Stream/mem_get_info/pinned memory 是同一套语义）。差异本质：NPU 侧的 H2D/D2H 走 CANN 的拷贝引擎与 HCCL 通路，CUDA 走 cudaMemcpyAsync + NCCL，runtime 不同但抽象层一一对应。一个语义陷阱：CUDA 的 pin_memory=True 走的是 host 端 page-locked memory + DMA，NPU 侧由 torch_npu 接管，名字一样但底层是 CANN 的 host pinned allocator——不要假设两侧行为完全一致，尤其是「pinned memory 是否真的异步」这点在 NPU 上要靠 non_blocking=True（:77/:79）+ 独立 stream 才能兑现。另外 CUDA 侧 offload 通常用 torch.cuda.mem_get_info 主动探显存余量决定要不要 offload，本文件没有这种探针——它按 block_idx < depth-1（:286）静态决定 offload 最后一层之前的所有激活，是「无脑全 offload」策略，不依显存余量动态调节。

行 17-50 · GetCnt：按 block_idx 给同层多张量编号 + 层间比例预取映射

class GetCnt:
    def __init__(self):
        self._block_idx = -1
        self._block_tensor_nums = {}  # offload tensors per block

    def get_cnt(self, block_idx):
        after_block = False
        if block_idx > self._block_idx:
            self._block_tensor_nums[block_idx] = 1
            if block_idx != 0:
                after_block = True
            self._block_idx = block_idx
        elif block_idx == self._block_idx:
            self._block_tensor_nums[block_idx] += 1
        else:
            # one step end
            self._block_idx = block_idx
            self._block_tensor_nums = {block_idx: 1}

        offload_tensor_key = "{}_{}".format(self._block_idx, self._block_tensor_nums[self._block_idx] - 1)
        return offload_tensor_key, after_block

    def get_prefetch_keys(self, block_idx, tensor_idx):
        prefetch_block_idx = max((idx for idx in self._block_tensor_nums.keys() if idx < block_idx), default=None)

        if prefetch_block_idx is None:
            return []

        prefetch_block_tensor_nums = self._block_tensor_nums[prefetch_block_idx]
        block_tensor_nums = self._block_tensor_nums[block_idx]
        start = tensor_idx * prefetch_block_tensor_nums // block_tensor_nums
        end = (tensor_idx + 1) * prefetch_block_tensor_nums // block_tensor_nums
        prefetch_idxs = list(range(start, end))
        return ["{}_{}".format(block_idx - 1, prefetch_idx) for prefetch_idx in prefetch_idxs]

GetCnt（:17）解决两个问题：(1) 给「同一层（block）内被 offload 的多个张量」分配稳定编号；(2) 反向时算「当前层第 i 个张量该预取上一层哪些张量」。_block_idx（:19）记住上一次见到的层号，_block_tensor_nums（:20）是 {block_idx: 本层已见张量数}。

:22-37 get_cnt(block_idx) 是个状态机，按传入 block_idx 与 self._block_idx 比较分三支：
- :24-28 block_idx 递增（进新层）：本层计数置 1，且若不是第 0 层则置 after_block=True（:27，标记「刚跨过一层」）；
- :29-30 block_idx 不变（同层多张量）：计数 +1；
- :31-34 block_idx 回退（一个 step 结束、新一轮前向开始）：重置计数表为新层单张量。
:36 生成 offload 键 "{block_idx}_{本层已见数-1}"，返回 (key, after_block)。after_block 被下游 _pack_to_cpu（:281）用来触发「把上一层残余 D2H 同步掉」（见 del_npu_tensor :209）。

:39-50 get_prefetch_keys(block_idx, tensor_idx) 是层间预取映射。:40 找出 block_idx 之前最近一层的 prefetch_block_idx（没有则 :42-43 返回空——第一层无可预取）。:45-48 用比例映射：把上一层张量数按 tensor_idx 在本层的占比切片，start = tensor_idx * 上层数 // 本层数、end = (tensor_idx+1) * 上层数 // 本层数，取 range(start, end)。:50 返回上一层那些下标的键 "{block_idx-1}_{i}"。语义是「本层第 i 个张量反向时，把上一层第 i 段那一坨张量从 host 预取回 device」——把下一层需要的激活提前在当前层反向时往回拷，掩盖 H2D 延迟。

诚实说：这个比例映射假设「相邻两层的激活数成比例且按序对应」——它不做内容匹配，纯靠下标区间。若两层张量数差异大（比如一层 2 个、下一层 8 个），8 个里每个都会预取 2 个中对应的同一片，可能重复拷贝。本文件没对此做去重，是已知简化。

class SwapTensor:
    def __init__(self, tensor, key):
        self.tensor = tensor
        self.size = tensor.size()
        self.storage_size = tensor.storage().size()
        self.tensor_cpu = torch.empty(tensor.shape, dtype=tensor.dtype, pin_memory=True, device='cpu')

        self.is_slice_tensor = tensor.storage().size() != tensor.numel()
        self.stat = "device"
        self.key = key

        self.h2d_event = torch_npu.npu.Event()

    # device to host
    def launch_d2h(self, stream):
        if self.stat != "device":
            return

        forward_event = torch_npu.npu.Event()
        forward_event.record()
        with torch.no_grad():
            with torch_npu.npu.stream(stream):
                stream.wait_event(forward_event)
                if self.is_slice_tensor:
                    self.tensor_cpu.copy_(self.tensor, non_blocking=True)
                else:
                    self.tensor_cpu.storage().copy_(self.tensor.storage(), non_blocking=True)
                self.stat = "host"

    # synchronize d2h and resize 0
    def wait_d2h_finished(self, stream, flag):
        if self.stat != "host":
            return
        if flag:
            torch_npu.npu.current_stream().wait_stream(stream)
            torch_npu.npu.default_stream().wait_stream(stream)
        self.tensor.storage().resize_(0)
        self.stat = "host"

左侧把 torch_npu.npu.Event/Stream/current_stream/default_stream 逐字换成 torch.cuda.*——这是 1:1 的语义对位。差异点有三处：
1. default_stream（:88）：NPU 侧显式 torch_npu.npu.default_stream().wait_stream(stream)——NPU 有独立的 default stream 概念，且这里要它也等 offload_stream；CUDA 侧通常不需要单独等 default stream（CUDA 的 default stream 即 legacy stream，行为不同），这一行在 CUDA 移植时往往可删或需重审。
2. storage().resize_(0) 的物理释放（:89）：两侧语义一致（把 storage 归零交还 allocator），但 NPU 的 caching allocator 与 CUDA 的 caching allocator 实现不同，碎片化行为有差异——NPU 上反复 resize_(0)/resize_(size) 可能触发 allocator 的碎片整理，长时间跑要注意 RSS 是否真的降了。
3. pinned memory（:58）：两侧都 pin_memory=True，但 CUDA 走 cudaHostAlloc+DMA，NPU 走 CANN host pinned allocator——名字一样、底层通路不同，non_blocking=True 是否真异步取决于各自 runtime 是否把它映射到独立 stream 的拷贝引擎。

class SingletonMeta(type):
    """
    single meta class.
    """

    _instances = {}

    def __call__(cls, *args, **kwargs):
        if cls not in cls._instances:
            instance = super().__call__(*args, **kwargs)
            cls._instances[cls] = instance

        return cls._instances[cls]

差异讲解

左侧把整份 NPU 实现逐行翻成 CUDA API：torch_npu.npu.* → torch.cuda.*，saved_tensors_hooks 是 torch 通用 API 两侧通用。差异与诚实结论：
1. 同流（offload_stream 同时当 h2d/d2h）：两侧调用方都这么写，导致 d2h_stream.wait_stream(h2d_stream)（:232）与 working_stream.wait_stream(h2d_stream)（:299）在多数情况下退化为空操作。理想的双流配置（h2d 一条、d2h 一条）能让 D2H 与 H2D 在不同流上重叠，掩盖拷贝延迟——本文件签名支持双流但调用方没启用，是次优。CUDA 侧若要复刻，建议把 :311/:312 拆成两条 torch.cuda.Stream() 才能吃到重叠收益。
2. custom_check_fn 过滤到 hidden_states（:315）：两侧语义一致——靠 data_ptr() 指针相等只挑主激活。CUDA 侧若要加更多张量（如 attention 的中间激活）只需放宽 lambda；本仓选择只 offload hidden_states 是激进保守策略，省 host 带宽但只换出每层一个张量的显存。
3. 训练才挂、推理不挂（:309 if self.training）：两侧同理——推理不需要反向，没激活要 save，挂钩子纯浪费。这个训练/推理分叉是通用设计，与硬件无关。
4. FSDP2 规模维度落点：本文件是「激活 offload」不是「参数 offload」。参数 offload 在 NVIDIA 侧是 FSDP(cpu_offload=OffloadPolicy())（走 fsdp_turbo :7 import 路径），本文件不碰参数；它叠加在参数已 sharded 的模型上，再省一层激活 HBM。两者换显存的对象不同、通路不同、代码也分家——这是理解本文件边界的最重要一点。

import torch
try:
    import torch_npu
except ImportError:
    pass
from torch.autograd.graph import saved_tensors_hooks


def base_check_fn(tensor):
    if isinstance(tensor._base, torch.nn.parameter.Parameter) or isinstance(tensor, torch.nn.parameter.Parameter):
        return False
    if tensor.storage().size() <= 0:
        return False
    return True


class GetCnt:
    def __init__(self):
        self._block_idx = -1
        self._block_tensor_nums = {}  # offload tensors per block

    def get_cnt(self, block_idx):
        after_block = False
        if block_idx > self._block_idx:
            self._block_tensor_nums[block_idx] = 1
            if block_idx != 0:
                after_block = True
            self._block_idx = block_idx
        elif block_idx == self._block_idx:
            self._block_tensor_nums[block_idx] += 1
        else:
            # one step end
            self._block_idx = block_idx
            self._block_tensor_nums = {block_idx: 1}

        offload_tensor_key = "{}_{}".format(self._block_idx, self._block_tensor_nums[self._block_idx] - 1)
        return offload_tensor_key, after_block

    def get_prefetch_keys(self, block_idx, tensor_idx):
        prefetch_block_idx = max((idx for idx in self._block_tensor_nums.keys() if idx < block_idx), default=None)

        if prefetch_block_idx is None:
            return []

        prefetch_block_tensor_nums = self._block_tensor_nums[prefetch_block_idx]
        block_tensor_nums = self._block_tensor_nums[block_idx]
        start = tensor_idx * prefetch_block_tensor_nums // block_tensor_nums
        end = (tensor_idx + 1) * prefetch_block_tensor_nums // block_tensor_nums
        prefetch_idxs = list(range(start, end))
        return ["{}_{}".format(block_idx - 1, prefetch_idx) for prefetch_idx in prefetch_idxs]


class SwapTensor:
    def __init__(self, tensor, key):
        self.tensor = tensor
        self.size = tensor.size()
        self.storage_size = tensor.storage().size()
        self.tensor_cpu = torch.empty(tensor.shape, dtype=tensor.dtype, pin_memory=True, device='cpu')

        self.is_slice_tensor = tensor.storage().size() != tensor.numel()
        self.stat = "device"
        self.key = key

        self.h2d_event = torch_npu.npu.Event()

    # device to host
    def launch_d2h(self, stream):
        if self.stat != "device":
            return

        forward_event = torch_npu.npu.Event()
        forward_event.record()
        with torch.no_grad():
            with torch_npu.npu.stream(stream):
                stream.wait_event(forward_event)
                if self.is_slice_tensor:
                    self.tensor_cpu.copy_(self.tensor, non_blocking=True)
                else:
                    self.tensor_cpu.storage().copy_(self.tensor.storage(), non_blocking=True)
                self.stat = "host"

    # synchronize d2h and resize 0
    def wait_d2h_finished(self, stream, flag):
        if self.stat != "host":
            return
        if flag:
            torch_npu.npu.current_stream().wait_stream(stream)
            torch_npu.npu.default_stream().wait_stream(stream)
        self.tensor.storage().resize_(0)
        self.stat = "host"

    # resize storage_size and host to device
    def launch_h2d(self, h2d_stream, flag, working_stream):
        if self.stat != "host":
            return
        backward_event = torch_npu.npu.Event()
        backward_event.record()
        if flag:
            self.tensor.storage().resize_(self.storage_size)
        with torch.no_grad():
            with torch_npu.npu.stream(h2d_stream):
                h2d_stream.wait_event(backward_event)
                if self.is_slice_tensor:
                    self.tensor.copy_(self.tensor_cpu, non_blocking=True)
                else:
                    self.tensor.storage().copy_(self.tensor_cpu.storage(), non_blocking=True)
                self.h2d_event.record()
                self.stat = "device"

                working_stream.wait_stream(h2d_stream)

    # resize storage_size and host to device
    def prefetch_launch_h2d(self, h2dstream, flag):
        if self.stat != "host":
            return
        backward_event = torch_npu.npu.Event()
        backward_event.record()
        if flag:
            self.tensor.storage().resize_(self.storage_size)
        with torch.no_grad():
            with torch_npu.npu.stream(h2dstream):
                h2dstream.wait_event(backward_event)
                if self.is_slice_tensor:
                    self.tensor.copy_(self.tensor_cpu, non_blocking=True)
                else:
                    self.tensor.storage().copy_(self.tensor_cpu.storage(), non_blocking=True)
                self.h2d_event.record()
                self.stat = "device"
                self.tensor.record_stream(h2dstream)

    # synchronize h2d
    def wait_h2d_finished(self):
        if self.stat != "device":
            return
        if self.h2d_event:
            torch_npu.npu.current_stream().wait_event(self.h2d_event)
            torch_npu.npu.default_stream().wait_event(self.h2d_event)
        self.stat = "device"


class SingletonMeta(type):
    """
    single meta class.
    """

    _instances = {}

    def __call__(cls, *args, **kwargs):
        if cls not in cls._instances:
            instance = super().__call__(*args, **kwargs)
            cls._instances[cls] = instance

        return cls._instances[cls]


class OffloadItem:
    """
    class for offload item
    """

    def __init__(self, act=None, ref_cnt=0, event=None):
        self.act = act
        self.ref_cnt = ref_cnt
        self.event = event

    def get_event(self):
        return self.event

    def has_event(self):
        return self.event is not None


class OffloadManager(metaclass=SingletonMeta):
    """
    class for offload manager
    """

    def __init__(self, check=False):
        self.items = {}
        self.check = check
        self.npu_item = []
        self.getcnt = GetCnt()

    def get_cnt(self, block_idx):
        return self.getcnt.get_cnt(block_idx)

    def assert_exist(self, key):
        if key not in self.items:
            raise RuntimeError(f"Key {key} does not exist in items")

    def exist(self, key):
        return key in self.items

    def assert_not_exist(self, key):
        if key not in self.items:
            raise RuntimeError(f"Key {key} already exist in items")

    def put(self, key, act, event=None):
        if key in self.items:
            self.items[key].act = act
            self.items[key].ref_cnt += 1
            self.items[key].event = event
        else:
            self.items[key] = OffloadItem(act, 1, event)

    def put_npu_tensor(self, act):
        self.npu_item.append(act)

    def del_npu_tensor(self, prefile_key, d2h_stream):
        for key in self.items.keys():
            if key.startswith(prefile_key):
                self.items[key].act.wait_d2h_finished(d2h_stream, True)

    def get(self, key):
        self.assert_exist(key)
        item = self.items[key]

        act = item.act
        if item.has_event():
            item.get_event().wait()

        item.ref_cnt -= 1
        if item.ref_cnt == 0:
            self.clear(key)
        return act

    def prefetch_get(self, block_idx, tensor_idx, h2d_stream, d2h_stream):
        prefetch_keys = self.getcnt.get_prefetch_keys(block_idx, tensor_idx)
        for prefetch_key in prefetch_keys:
            if self.exist(prefetch_key):
                prefetch_swap_tensor = self.get(prefetch_key)
                d2h_stream.wait_stream(h2d_stream)
                prefetch_swap_tensor.prefetch_launch_h2d(h2d_stream, True)
                prefetch_swap_tensor.tensor.record_stream(h2d_stream)

    def empty(self):
        return len(self.items) == 0

    def clear(self, key=None):
        if key is None:
            self.items.clear()
        else:
            self.assert_exist(key)
            self.items.pop(key)

    # event interface #

    def get_event(self, key):
        self.assert_exist(key)
        item = self.items[key]
        event = item.get_event()
        return event

    def has_event(self, key):
        if not self.exist(key):
            return False
        item = self.items[key]
        return item.has_event()
